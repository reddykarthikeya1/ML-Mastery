# Phase 3: Core ML & Deep Learning

## 📋 Overview

**Duration:** Months 8-12
**Prerequisites:** Phase 1 (Mathematics), Phase 2 (Python DSA)

Phase 3 covers classical machine learning algorithms and deep learning - the core skills for becoming an ML engineer.

**Note:** ML Fundamentals (Section 6) is covered in Phase 2. Phase 3 starts from Section 7.

```
Phase 3 Structure:
├── 00-ML-Fundamentals (Section 6 - from Phase 2)
├── 01-Classical-ML-Algorithms (Section 7)
├── 02-Deep-Learning-Fundamentals (Section 8)
└── 03-Deep-Learning-Frameworks (Section 9)
```

---

## 📚 Section Breakdown

### 00-ML-Fundamentals (Section 6)
- Introduction to Machine Learning
- Supervised Learning (Regression, Classification)
- Unsupervised Learning (Clustering, Dimensionality Reduction)
- Model Evaluation and Validation
- Feature Engineering

### 01-Classical-ML-Algorithms (Section 7)
- **7.1 Linear Models** - Linear/Logistic Regression, GLM, Ridge/Lasso/Elastic Net
- **7.2 Tree-Based Models** - Decision Trees, Random Forest, XGBoost, LightGBM, CatBoost
- **7.3 Instance-Based Learning** - KNN, Distance Metrics, KD-Trees, LVQ
- **7.4 Support Vector Machines** - Maximum Margin, Kernels, SVR, SMO
- **7.5 Naive Bayes** - Bayes Theorem, Gaussian/Multinomial/Bernoulli NB
- **7.6 Clustering Algorithms** - K-Means, DBSCAN, Hierarchical, GMM, EM
- **7.7 Dimensionality Reduction** - PCA, LDA, t-SNE, UMAP, Manifold Learning

### 02-Deep-Learning-Fundamentals (Section 8)
- **8.1 Neural Network Basics** - Perceptron, MLP, Activation Functions, Loss Functions, Backpropagation
- **8.2 Optimization Algorithms** - GD Variants, Momentum, Adaptive Methods, LR Scheduling
- **8.3 Regularization Techniques** - L1/L2, Dropout, BatchNorm, LayerNorm, Data Augmentation
- **8.4 Training Deep Networks** - Initialization, Vanishing Gradients, Debugging
- **8.5 Convolutional Neural Networks** - CNN Fundamentals, Architectures (LeNet to EfficientNet)
- **8.6 Recurrent Neural Networks** - RNN, LSTM, GRU, Bidirectional RNNs

### 03-Deep-Learning-Frameworks (Section 9)
- **9.1 TensorFlow/Keras** - Tensors, Keras APIs, Training, tf.data, Deployment
- **9.2 PyTorch** - Tensors, Autograd, nn.Module, Training Loops, DataLoader
- **9.3 JAX** - jit, grad, vmap, pmap, Flax, Optax

## 🎯 Learning Outcomes

After completing Phase 3:

✅ **Classical ML**
- Implement and tune classical ML algorithms from scratch
- Understand when to use each algorithm
- Handle real-world datasets with proper preprocessing
- Evaluate and compare models using appropriate metrics

✅ **Deep Learning**
- Understand neural network architectures from basics to advanced
- Build and train deep neural networks from scratch
- Implement CNNs for image tasks and RNNs for sequence tasks
- Apply regularization and optimization techniques effectively

✅ **Production Ready**
- Use TensorFlow/Keras for production deployment
- Use PyTorch for research and prototyping
- Understand JAX for high-performance computing
- Deploy ML models using appropriate frameworks
